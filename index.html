About this project — Leap is built with HTML, CSS, and JavaScript. Under the hood, it uses a Singly Linked List for dynamic task storage, a Stack for undo functionality, and Bubble Sort for priority ordering — all implemented from scratch without any libraries. Data is saved locally on your device using localStorage.
